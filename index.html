Background AI System

Sistem Artificial Intelligence (AI) ini digunakan untuk meningkatkan kualitas inspeksi pada line produksi secara otomatis dan real-time dengan metode object detection untuk menentukan kondisi OK atau NG.

Seluruh sistem terintegrasi dengan PLC melalui jaringan LAN, dimana proses dimulai dari trigger PLC yang akan mengaktifkan kamera dan sistem pada Raspberry Pi. Tanpa trigger dari PLC, sistem tidak akan berjalan dan kamera tidak aktif.

Gambar yang ditangkap akan diproses menggunakan model AI, dan pada sistem detection dibantu oleh Coral USB Accelerator.

Hasil ditampilkan secara real-time melalui perangkat monitoring seperti tablet atau monitor.

PLC → Camera → AI → Display
